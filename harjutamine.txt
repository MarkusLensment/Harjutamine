Github on veebisait kus programeerijad saavad kirjutada, salvestaja ja teiste kasutajatega enda koodi jagada.
Githubis saab enda koodi kopeerida ja seda erinevates kaustades säilitada.
See annab kasutajale võimaluse osa enda koodist muuta ja seda katsetada, seejuures ülejäänud koodi puutumata. 
Kui muudetud kood töötab ootuspäraselt siis laseb Github seda lihtsasti põhilise koodiga ühendada. 

Mina saan githubi kasutada, salvestades sinna enda loodud projekte.
Saan seal ka enda projekte muuta, seejuures kartmata, et see midagi muud ära lõhuks, või et minu esialgne kood kaduma läheks.

Githubi ajalugu

Githubi hakati välja töötama 2007 aasta oktoobris, kasutajate jõudis see juba lärgmisel aastal. 
Githubi asutajateks olid Tom Preston-Werner, Chris Wanstrath, P.J. Hyett ja Scott Chancon. 
Algselt oli Github starupilku mõtlemisega organisatsioon kus kõikidele töötajatele anti võimalus veebilehte oma soovitud suunas edasi arendada. Hiljem aga sai firma siiski tüüpilisema juhtimisstruktuuri. 2018 ostis Microsoft firma ära.
